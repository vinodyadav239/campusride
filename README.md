# CampusRide 🛺

Smart auto-booking platform for college students built with MERN stack.

## Problem
Students at our college face auto shortage during festivals, 
no rides at night, and unfair pricing with no transparency.

## Solution
CampusRide connects students with auto drivers in real time 
with fair dynamic pricing and a safety system.

## Features
- 🛺 Book pool/solo/scheduled rides
- 💰 Transparent fare table with surge pricing
- 🔄 Real-time ride tracking via Socket.IO  
- 🌙 Night ride support with driver incentives
- 🚨 Emergency alert system — alerts police + parents
- 📞 Fake call feature for student safety
- ↩ Bidirectional routes (college ↔ destinations)

## Tech Stack
- **Frontend**: React PWA
- **Backend**: Node.js + Express
- **Database**: MongoDB
- **Real-time**: Socket.IO
- **Auth**: JWT

## Setup
### Backend
cd server
npm install
npm run dev

### Frontend  
cd client
npm install
npm start
